# ScientificState Trademark Policy

## 1. Trademark Scope

The following are trademarks of ScientificState Contributors:

- The name **"ScientificState"**
- The name **"ScientificState Verified"**
- The ScientificState logo and visual identity

These trademarks are separate from the Apache-2.0 software license.
The code is open source; the brand is protected.

## 2. Permitted Use

You MAY:

- Use the ScientificState source code under the Apache-2.0 license
- Create derivative works (forks, modifications, extensions)
- State factual compatibility: "Compatible with ScientificState v0.8"
- Reference the project: "Built using ScientificState framework"
- Use for academic citation and research reference

## 3. Restricted Use

You MAY NOT:

- Name your product or service "ScientificState" or any confusingly
  similar name (e.g., "SciState", "Scientific-State", "ScientificState")
- Use "ScientificState Verified" to describe outputs that have not
  been verified through the official ScientificState trust network
- Use the ScientificState logo in your product branding without
  written permission
- Imply official endorsement, sponsorship, or affiliation with the
  ScientificState project without written agreement

## 4. Fork Naming

If you fork ScientificState, you MUST:

- Choose a distinct name for your distribution
- Remove ScientificState branding (logo, "Verified" badges)
- Clearly state that your project is a fork, not the original

Example of compliant fork naming:
- "MyResearchTool (based on ScientificState)" — OK
- "MyResearchTool — ScientificState Edition" — NOT OK

## 5. Trust Tier Labels

The following trust tier labels have specific meanings within the
ScientificState ecosystem:

| Label | Meaning |
|-------|---------|
| `local_unverified` | Computed locally, no external verification |
| `community_signed` | Verified by another user/instance |
| `scientificstate_verified` | Verified through official federation network |

Only results that have passed through the official ScientificState
federation network and institutional attestation process may carry
the `scientificstate_verified` label. Forks and independent
deployments MUST NOT use this label for their own verification
processes.

## 6. Reporting Violations

If you believe someone is misusing the ScientificState trademark,
please report it via the process described in SECURITY.md.
